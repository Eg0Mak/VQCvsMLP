# VQC vs MLP

## О проекте

Этот проект исследует **сравнение Variational Quantum Classifier (VQC)** и **классической нейросети (MLP)** на простых задачах классификации.  
Основная цель - проверить, насколько квантовая модель может конкурировать с классическими методами машинного обучения на малых и средних датасетах.

## Идея

Вариационные квантовые классификаторы часто называют квантовым аналогом нейросетей.  
Интересно проверить:
- Дает ли VQC преимущество по точности и обобщающей способности?  
- Как ведет себя VQC на простых и сложных датасетах (например, circles / moons)?  

## Эксперименты

1. Обучение VQC на синтетическом датасете.
2. Обучение MLP с сопоставимым числом параметров.
3. Сравнение метрик точности (accuracy).
4. Визуализация границ решений обеих моделей.

## Технологии

- [PennyLane](https://github.com/PennyLaneAI/pennylane?ysclid=mlcur65i9n159127497) (квантовое машинное обучение)
- PyTorch (классическая нейросеть)
- scikit-learn (датасеты и метрики)
- Matplotlib / Seaborn (визуализация)
